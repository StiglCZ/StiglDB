# StiglDB

### LightWeight database system made in c#


### Query system
`command—arg1—arg2—arg3—...`


### Commands

Create table:  c—<tableName>
#### Select from table:
`s—<tableIndex>—<tableColumn>—<searchQuery>—<searchQuery>`
#### Insert to table(Not done yet correctly):
`i—<tableIndex>—<row content>`
#### Get table list:
`g`
